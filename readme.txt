
This is a stock management system that allows users to insert the product that he/she wishes to add.
Allows the user to type their name to create the user ID.
Allows the user to insert the four types of products into the system.
Allows the user to view the contents of the created products.
Allows the user to add stock of the created product.
Allows the user to deduct stock of the created product.
Allows the user to set the status of the product.
